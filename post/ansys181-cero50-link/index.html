ANSYS 18.1与Cero 5.0文件关联的方法

2019-07-30 Views 教程

马上念研究生，用到SolidWorks、Cero和ANSYS，建模完要仿真分析，这时候需要将SolidWorks或者Cero画好的模型导入到ANSYS当中。SolidWorks我安装的版本是2016，ANSYS我安装的是18.1版本。这样SW和ANSYS创建链接的时候没有任何问题，但是我使用的Cero是5.0版本的，ANSYS不支持这么高版本的Cero，不能从软件内直接创建链接。这时候我们就需要手动操作来创建链接。

首先打开路径：
D:\Program Files\ANSYS Inc\v181\aisol\CADIntegration\ProE\ProEPages\config

找到文件WBPlugInPE.dat

用记事本打开这个文件，文件内容如下：

# Under normal conditions this file is automatically generated by the CAD Configuration Manager.  This DAT file 
# is for example purposes, to illistrate how to construct the file to manually register the ANSYS Plug-In to Creo Parametric. 

# (1)  Registration of ANSYS Associative interface to Creo Parametric.  Adjust exec_path/text_dir to match your ANSYS installation path

NAME WB181PluginProWF
EXEC_FILE C:\Program Files\ANSYS Inc\v181\aisol\CADIntegration\ProE\winx64\WBPlugInCreo3COM.dll
TEXT_DIR  C:\Program Files\ANSYS Inc\v181\aisol\CADIntegration\ProE\ProEPages\Language\en-us
STARTUP dll
delay_start FALSE
allow_stop TRUE
REVISION ProEWildfire
END

# (2) Registration of MAPDL interface.  Adjust exec_path/text_dir to match your ANSYS installation path

NAME ac4pro181dll
exec_path  C:\Program Files\ANSYS Inc\v181\ANSYS\ac4\bin\pro\winx64\ac4pro.exe
text_path  C:\Program Files\ANSYS Inc\v181\ANSYS\ac4\data\pro\text
STARTUP dll
delay_start FALSE
allow_stop TRUE
unicode_encoding FALSE
revision 24.0
end

将其中的内容修改为：

# Under normal conditions this file is automatically generated by the CAD Configuration Manager.  This DAT file 
# is for example purposes, to illistrate how to construct the file to manually register the ANSYS Plug-In to Creo Parametric. 

# (1)  Registration of ANSYS Associative interface to Creo Parametric.  Adjust exec_path/text_dir to match your ANSYS installation path

NAME WB181PluginProWF
EXEC_FILE D:\Program Files\ANSYS Inc\v181\aisol\CADIntegration\ProE\winx64\WBPlugInCreo3COM.dll
TEXT_DIR  D:\Program Files\ANSYS Inc\v181\aisol\CADIntegration\ProE\ProEPages\Language\en-us
STARTUP dll
delay_start FALSE
allow_stop TRUE
REVISION ProEWildfire
END

# (2) Registration of MAPDL interface.  Adjust exec_path/text_dir to match your ANSYS installation path

NAME ac4pro181dll
exec_path  D:\Program Files\ANSYS Inc\v181\ansys\ac4\bin\pro\winx64\ac4pro.exe
text_path  D:\Program Files\ANSYS Inc\v181\ansys\ac4\data\pro\text
STARTUP dll
delay_start FALSE
allow_stop TRUE
unicode_encoding FALSE
revision 24.0
end

主要修改了其中(1)中的以下内容：

EXEC_FILE D:\Program Files\ANSYS Inc\v181\aisol\CADIntegration\ProE\winx64\WBPlugInCreo3COM.dll
TEXT_DIR  D:\Program Files\ANSYS

和(2)中的以下内容：

exec_path  D:\Program Files\ANSYS Inc\v181\ansys\ac4\bin\pro\winx64\ac4pro.exe
text_path  D:\Program Files\ANSYS Inc\v181\ansys\ac4\data\pro\text

如果只修改(1)而不修改（2）的话，最后配置出来的Cero打开会报错。

然后到Cero中，找到以下路径：
D:\PTC\Creo 5.0.0.0\Common Files\text

找到其中的config.pro文件。

用记事本打开，内容如下：

drawing_setup_file $PRO_DIRECTORY\text\prodetail.dtl
format_setup_file $PRO_DIRECTORY\text\prodetail.dtl
pro_unit_length unit_inch
pro_unit_mass unit_pound
template_designasm $PRO_DIRECTORY\templates\inlbs_asm_design.asm
template_new_ecadasm $PRO_DIRECTORY\templates\inlbs_ecad_asm.asm
template_drawing $PRO_DIRECTORY\templates\c_drawing.drw
template_sheetmetalpart $PRO_DIRECTORY\templates\inlbs_part_sheetmetal.prt
template_solidpart $PRO_DIRECTORY\templates\inlbs_part_solid.prt
template_boardpart $PRO_DIRECTORY\templates\inlbs_ecad_board.prt
todays_date_note_format %Mmm-%dd-%yy
tolerance_standard ansi
weld_ui_standard ansi
search_path_file $CREO_COMMON_FILES\afx\parts\prolibrary\search.pro

向其中最后一行后新建一行，添加以下内容：

PROTKDAT D:\Program Files\ANSYS Inc\v181\aisol\CADIntegration\ProE\ProEPages\config\WBPlugInPE.dat

添加完成后，变成如下内容：

drawing_setup_file $PRO_DIRECTORY\text\prodetail.dtl
format_setup_file $PRO_DIRECTORY\text\prodetail.dtl
pro_unit_length unit_inch
pro_unit_mass unit_pound
template_designasm $PRO_DIRECTORY\templates\inlbs_asm_design.asm
template_new_ecadasm $PRO_DIRECTORY\templates\inlbs_ecad_asm.asm
template_drawing $PRO_DIRECTORY\templates\c_drawing.drw
template_sheetmetalpart $PRO_DIRECTORY\templates\inlbs_part_sheetmetal.prt
template_solidpart $PRO_DIRECTORY\templates\inlbs_part_solid.prt
template_boardpart $PRO_DIRECTORY\templates\inlbs_ecad_board.prt
todays_date_note_format %Mmm-%dd-%yy
tolerance_standard ansi
weld_ui_standard ansi
search_path_file $CREO_COMMON_FILES\afx\parts\prolibrary\search.pro
PROTKDAT D:\Program Files\ANSYS Inc\v181\aisol\CADIntegration\ProE\ProEPages\config\WBPlugInPE.dat

这样，保存以后重新打开Cero 5.0就可以看到新的ANSYS工具菜单了。
