Centos7下安装构造Aria2下载器

2019-05-18 Views 教程

在前面的文章中，有aria2下载器和oneindex自动上传一键脚本，固然好用，但由于时常遇到下载后需要对一些文件进行处理，比如删除某些文件、调整文件目录结构及解压等操作，就不能要求自动上传。因此，我重新找了一个脚本，以满足日常使用的需要。另外由于日常解压等操作，将其他一些软件安装代码也贴出来，方便使用。

基于镜像：gingko/centos6.8
该镜像支持中文，但有时候解压.7z还是会出现乱码问题。

预配置：

yum update -y && yum update -y && yum install -y wget screen unzip vim curl

这是我自己修改的CCAA一键脚本，仅适用于Centos系统：

wget http://dl.chyon.cn/Aria2/CCAA/ccaa.zip
unzip ccaa.zip && cd ccaa && sh ccaa.sh

官方的CCAA系统请参考Github。

安装unrar：

cd /usr/local
wget http://www.rarlab.com/rar/rarlinux-x64-5.3.0.tar.gz
tar -xzvf rarlinux-x64-5.3.0.tar.gz
ln -s /usr/local/rar/rar /usr/local/bin/rar
ln -s /usr/local/rar/unrar /usr/local/bin/unrar

使用方法：

unrar x abc.rar

安装7z：

wget https://sourceforge.net/projects/p7zip/files/p7zip/16.02/p7zip_16.02_src_all.tar.bz2
yum install -y bzip2
tar -jxvf p7zip_16.02_src_all.tar.bz2
cd p7zip_16.02
yum install gcc -y
yum install gcc-c++ -y
make && make install

使用方法：

7za x abc.rar
