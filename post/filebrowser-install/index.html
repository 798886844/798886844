FileBrowser安装与使用

2018-06-05 Views 教程 分享 软件

搭建环境和前面完全一样，也是Docker环境，镜像：rastasheep/ubuntu-sshd。

直接安装，系统环境为：

apt-get update
apt-get install -y wget unzip screen

在此基础之上直接安装FileBrowser即可：

2019/5/13 更新：
新版本的filebrowser不知道抽什么疯，删了原来的版本包，换了新的包，而且改了一些默认参数。所以我替换了下载源，改成我的下载站了。

cd /root
mkdir filebrowser
cd filebrowser
wget https://github.com/filebrowser/filebrowser/releases/download/v2.0.12/linux-amd64-filebrowser.tar.gz
tar -zxvf linux-amd64-filebrowser.tar.gz

然后用screen启动：

screen -dmS filebrowser ./filebrowser --port 82 --address 0.0.0.0 --root /root/Download

其中：82是端口号，可以任意修改，--root后面是管理目录，可以任意修改。

然后访问http://ip:82/即可查看你的FileBrowser，默认账号和密码都是admin。
