Debian一键安装Aria2+AriaNG+Rclone+DirectoryLister

2018-11-22 Views 教程

构建环境：hklcf/debian-ssh-docker

预配置：

解析两个域名到你的VPS，注意是两个。

apt-get update -y && apt-get upgrade -y && apt-get install wget cron -y

一键安装：

wget -O Aria2_Rc_D.sh http://dl.chyon.cn/?/Aria2/Aria2_Rc_D.sh && bash Aria2_Rc_D.sh

然后脚本安装完成后，就可以直接通过Aria2Ng面板连接Aria2下载文件，并通过网盘观看了。

相关目录

Aria2下载目录：/root/Download

DirectoryLister目录：/home/wwwroot/a.xx.com

Aria2Ng目录：/home/wwwroot/b.xx.com

挂载目录：/home/wwwroot/a.xx.com/Cloud
